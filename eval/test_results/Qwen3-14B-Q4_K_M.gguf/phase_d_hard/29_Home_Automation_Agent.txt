### **Identified Logical Conflicts and Resolutions**

1. **Conflict 1: Hallway Lights (5% vs Full Brightness)**  
   - **Conflict**: The initial "movie night" command says to dim *all* lights to 5%, but the same command also specifies keeping hallway lights at full brightness for safety.  
   - **Resolution**: Override the general "dim all lights" rule for the hallway. During movie night, explicitly set hallway lights to full brightness, while dimming all other lights to 5%.  

2. **Conflict 2: Smoke Detector Priority (Lights to 100% vs Movie Night Lights at 5%)**  
   - **Conflict**: If a smoke detector triggers, all automations must stop, and lights must go to 100%. This directly contradicts the movie night setting of dimming lights to 5%.  
   - **Resolution**: Prioritize the smoke detector rule. If triggered, immediately halt all automations, set all lights to 100%, and disable other actions (e.g., projector, sound system).  

3. **Conflict 3: "Do Not Disturb" Mode vs 2-Hour Inactivity Rule**  
   - **Conflict**: "Do not disturb" mode is set during movie night, but the 2-hour inactivity rule requires turning everything off if no movement is detected.  
   - **Resolution**: The 2-hour inactivity rule takes precedence. If no movement is detected for 2 hours, override the "do not disturb" mode and turn off all systems, including deactivating the projector, sound system, cameras, and unlocking doors.
