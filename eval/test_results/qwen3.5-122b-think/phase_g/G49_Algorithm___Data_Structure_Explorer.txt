2.  **Design Decisions:**
    *   **Data Structure:** To achieve O(1) `get` and `put` with LRU ordering, `collections.OrderedDict` is the standard Pythonic choice. It maintains insertion order and allows O(1) move-to-end and popitem.
    *   **Expiration:** Each entry needs to store a value, a timestamp (expiration time), and potentially the TTL.
    *   **Lazy Removal:** When accessing (`get`, `put` potentially), check if the current time > expiration time. If so, remove it and update stats.
    *   **Stats:** Need counters for hits, misses, evictions, expirations.
    *   **Capacity:** When `put` exceeds capacity, remove the least recently used item (oldest in OrderedDict).
    *   **TTL:** `None` means infinite. Otherwise, `time.time() + ttl`.

        *   If exists, check expiration.
        *   If expired: remove, update stats (expiration), return None.
        *   If not expired: move to end (MRU), update stats (hit), return value.
        *   If not exists: update stats (miss), return None.
    *   `put(key, value, ttl)`:

If `put` is called on an existing key, we just update it. If `put` is called on a new key and cache is full, we evict the LRU.
        *   However, if the LRU item is expired, should we evict it? Yes, it's the LRU.
        *   Let's refine `put`:
            1. If key in cache:
                2. Check expiration. If expired, remove it (count expiration), then proceed to insert new.
                3. If not expired, move to end, update value/ttl.
            4. If key not in cache:

                    7. To keep O(1), we just pop the first item. We should count it as an eviction.
                8. Insert new item.
    *   **Refinement on Expiration Logic:**
        *   The requirement says "Expired entries should be lazily removed on access".
        *   `get` is access. `put` is access.
        *   If `get` finds an expired item, remove it, count expiration, return None.
        *   If `put` finds an existing item, update it. If the existing item was expired, we overwrite it anyway.
        *   If `put` needs to evict (capacity full), we pop the LRU. Should we check if the popped LRU is expired?
            *   Strictly speaking, if it's expired, it's garbage. But since we are lazy, it sits there. When we evict it to make room, we remove it. Does that count as eviction or expiration?
            *   Usually, eviction is capacity-driven. Expiration is time-driven.
            *   If I pop an expired item to make room, it's technically an eviction triggered by capacity, but the item was expired.
            *   Let's stick to standard behavior: `popitem` counts as eviction. If `get` finds an expired item, it counts as expiration.
            *   Wait, if `put` overwrites an existing key, does it count as eviction? No.
            *   If `put` adds a new key and removes an old one, that's eviction.

import time
from collections import OrderedDict

class LRUCache:
    def __init__(self, capacity: int):
        self.capacity = capacity
        self.cache = OrderedDict()
        self.stats = {"hits": 0, "misses": 0, "evictions": 0, "expirations": 0}

    def _is_expired(self, expire_time):
        if expire_time is None:
            return False
        return time.time() > expire_time

    def get(self, key: str) -> any:
        if key not in self.cache:
            self.stats["misses"] += 1
            return None

        value, expire_time = self.cache[key]
        if self._is_expired(expire_time):
            del self.cache[key]
            self.stats["expirations"] += 1
            self.stats["misses"] += 1
            return None

        self.cache.move_to_end(key)
        self.stats["hits"] += 1
        return value

    def put(self, key: str, value: any, ttl: float = None):
        expire_time = None
        if ttl is not None:
            expire_time = time.time() + ttl

        if key in self.cache:
            _, old_expire = self.cache[key]
            if self._is_expired(old_expire):
                del self.cache[key]
                self.stats["expirations"] += 1
            else:
                self.cache.move_to_end(key)
            self.cache[key] = (value, expire_time)
        else:
            if len(self.cache) >= self.capacity:
                _, _ = self.cache.popitem(last=False)
                self.stats["evictions"] += 1
            self.cache[key] = (value, expire_time)

    def delete(self, key: str) -> bool:
        if key in self.cache:
            del self.cache[key]
            return True
        return False

    def size(self) -> int:
        count = 0
        for _, (_, expire_time) in self.cache.items():
            if not self._is_expired(expire_time):
                count += 1
        return count

    def get_stats(self) -> dict:
        return self.stats.copy()

    def keys(self) -> list:
        valid_keys = []
        for key, (_, expire_time) in self.cache.items():
            if not self._is_expired(expire_time):
                valid_keys.append(key)
        return valid_keys[::-1]

    def clear(self) -> None:
        self.cache.clear()
        self.stats = {"hits": 0, "misses": 0, "evictions": 0, "expirations": 0}
