### Key Features:
1. **6+ Agents**: Includes `Triage_Validator`, `Fraud_Checker`, `Approval_Manager`, `Human_Escalator`, `Payment_Processor`, and `Notification` agents.
2. **Validation/Fraud Check**: `Triage_Validator` → `Fraud_Checker` (fraud flagged → escalation).
3. **Approval/Payment**: `Approval_Manager` → `Payment_Processor` (only if approved).
4. **Conditional Branching**: Fraud flagged → `Human_Escalator`; approval → `Payment_Processor`.
5. **Fallbacks**: Each agent has retry/escalation logic.
6. **Audit Logging**: Implicit in `output` fields (e.g., `audit_log` in `Notification_Success`).
7. **Human Escalation**: `Fraud_Checker` → `Human_Escalator` for edge cases.
8. **Timeouts**: Summed to < 60 seconds (e.g., `Triage_Validator` + `Fraud_Checker` = 25s).
