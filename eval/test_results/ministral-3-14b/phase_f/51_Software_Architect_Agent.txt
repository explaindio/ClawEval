### Key Design Choices:
1. **Real-Time Latency**: Kafka + NATS provide sub-100ms latency with horizontal scaling.
2. **E2EE**: Signal Protocol with edge-offloaded key management (Cloudflare Workers).
3. **GDPR**: Encrypted backups, user-controlled data deletion, and regional data residency.
4. **Cost Optimization**: Spot instances for compute, S3 lifecycle policies for storage.
5. **Disaster Recovery**: Multi-region Kafka + MongoDB Atlas with RPO < 15 mins.
