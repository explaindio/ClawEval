## Party Problem

Let:

* C = Set of people who drink coffee (|C| = 90)
* T = Set of people who drink tea (|T| = 80)
* J = Set of people who drink juice (|J| = 70)
* U = Total number of people at the party (|U| = 100)

We want to find the minimum value of |C ∩ T ∩ J|.

Using the Principle of Inclusion-Exclusion:

|C ∪ T ∪ J| = |C| + |T| + |J| - |C ∩ T| - |C ∩ J| - |T ∩ J| + |C ∩ T ∩ J|

We know that |C ∪ T ∪ J| ≤ |U| = 100 (the total number of people).  We need to find the *minimum* |C ∩ T ∩ J|.  To do this, we want to *maximize* |C ∪ T ∪ J|.  The maximum value of |C ∪ T ∪ J| is 100.

Therefore:

100 ≥ 90 + 80 + 70 - |C ∩ T| - |C ∩ J| - |T ∩ J| + |C ∩ T ∩ J|
100 ≥ 240 - |C ∩ T| - |C ∩ J| - |T ∩ J| + |C ∩ T ∩ J|
|C ∩ T| + |C ∩ J| + |T ∩ J| - |C ∩ T ∩ J| ≥ 140

Now, we need to find lower bounds for the intersections.

* |C ∩ T| ≥ |C| + |T| - |U| = 90 + 80 - 100 = 70
* |C ∩ J| ≥ |C| + |J| - |U| = 90 + 70 - 100 = 60
* |T ∩ J| ≥ |T| + |J| - |U| = 80 + 70 - 100 = 50

Substituting these minimum values into the inequality:

70 + 60 + 50 - |C ∩ T ∩ J| ≥ 140
180 - |C ∩ T ∩ J| ≥ 140
|C ∩ T ∩ J| ≤ 40

However, this gives us an *upper* bound on |C ∩ T ∩ J|.  We need to find the *minimum*.

Let's consider the complements.

* |C'| = 10 (people who don't drink coffee)
* |T'| = 20 (people who don't drink tea)
* |J'| = 30 (people who don't drink juice)

|C' ∪ T' ∪ J'| ≤ 100

Using Inclusion-Exclusion again:

|C' ∪ T' ∪ J'| = |C'| + |T'| + |J'| - |C' ∩ T'| - |C' ∩ J'| - |T' ∩ J'| + |C' ∩ T' ∩ J'|
|C' ∪ T' ∪ J'| = 10 + 20 + 30 - |C' ∩ T'| - |C' ∩ J'| - |T' ∩ J'| + |C' ∩ T' ∩ J'|
|C' ∪ T' ∪ J'| = 60 - |C' ∩ T'| - |C' ∩ J'| - |T' ∩ J'| + |C' ∩ T' ∩ J'|

Since |C' ∪ T' ∪ J'| ≤ 100, we have:

60 - |C' ∩ T'| - |C' ∩ J'| - |T' ∩ J'| + |C' ∩ T' ∩ J'| ≤ 100
-|C' ∩ T'| - |C' ∩ J'| - |T' ∩ J'| + |C' ∩ T' ∩ J'| ≤ 40
|C' ∩ T' ∩ J'| ≤ 40 + |C' ∩ T'| + |C' ∩ J'| + |T' ∩ J'|

We want to minimize |C ∩ T ∩ J|.  By De Morgan's Law:

|C ∩ T ∩ J| = |(C' ∪ T' ∪ J')'| = |U| - |C' ∪ T' ∪ J'|
|C ∩ T ∩ J| = 100 - |C' ∪ T' ∪ J'|

To minimize |C ∩ T ∩ J|, we need to maximize |C' ∪ T' ∪ J'|.  The maximum value of |C' ∪ T' ∪ J'| is 100.

|C' ∪ T' ∪ J'| = 10 + 20 + 30 - |C' ∩ T'| - |C' ∩ J'| - |T' ∩ J'| + |C' ∩ T' ∩ J'|
To maximize this, we want to minimize the intersections.  The minimum value for each intersection is 0.

So, |C' ∪ T' ∪ J'| = 60.

|C ∩ T ∩ J| = 100 - 60 = 40

However, this is incorrect. Let's go back to the original inclusion-exclusion.

100 ≥ 90 + 80 + 70 - |C ∩ T| - |C ∩ J| - |T ∩ J| + |C ∩ T ∩ J|
100 ≥ 240 - |C ∩ T| - |C ∩ J| - |T ∩ J| + |C ∩ T ∩ J|
|C ∩ T ∩ J| ≥ 140 - |C ∩ T| - |C ∩ J| - |T ∩ J|

To minimize |C ∩ T ∩ J|, we need to maximize |C ∩ T|, |C ∩ J|, and |T ∩ J|.

* |C ∩ T| ≤ min(|C|, |T|) = 80
* |C ∩ J| ≤ min(|C|, |J|) = 70
* |T ∩ J| ≤ min(|T|, |J|) = 70

|C ∩ T ∩ J| ≥ 140 - 80 - 70 - 70 = 140 - 220 = -80. This doesn't help.

Let's use the following:

|C ∩ T| = |C| + |T| - |C ∪ T| ≥ 90 + 80 - 100 = 70
|C ∩ J| = |C| + |J| - |C ∪ J| ≥ 90 + 70 - 100 = 60
|T ∩ J| = |T| + |J| - |T ∪ J| ≥ 80 + 70 - 100 = 50

|C ∩ T ∩ J| = |C ∩ T| + |J| - |(C ∩ T) ∪ J| ≥ 70 + 70 - 100 = 40
|C ∩ T ∩ J| = |C ∩ J| + |T| - |(C ∩ J) ∪ T| ≥ 60 + 80 - 100 = 40
|C ∩ T ∩ J| = |T ∩ J| + |C| - |(T ∩ J) ∪ C| ≥ 50 + 90 - 100 = 40

Therefore, the minimum number of people who drink all three is **40**.
