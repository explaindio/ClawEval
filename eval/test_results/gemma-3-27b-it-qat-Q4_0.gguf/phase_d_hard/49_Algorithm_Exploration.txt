Design an algorithm for this problem. There's an O(n) solution but most people give O(n log n) or O(n²). Find the optimal one.

Problem: Given an array of n integers, find the maximum difference arr[j] - arr[i] such that j > i (the larger element must come AFTER the smaller element). If no positive difference exists, return 0.

Example: [2, 3, 10, 6, 4, 8, 1] → Answer: 8 (10-2)
Example: [7, 9, 5, 6, 3, 2] → Answer: 2 (9-7, not 5-2 because 5 comes after 9)
Example: [10, 8, 6, 4, 2] → Answer: 0 (strictly decreasing)

4. Prove the O(n) solution is correct with a formal argument

def max_difference_naive(arr):
    """
    Naive O(n^2) solution.

    Iterates through all possible pairs (i, j) where j > i and calculates the difference arr[j] - arr[i].
    Keeps track of the maximum difference found.

    Args:
        arr: The input array of integers.

    Returns:
        The maximum difference arr[j] - arr[i] such that j > i, or 0 if no positive difference exists.
    """
    n = len(arr)
    max_diff = 0
    for i in range(n):
        for j in range(i + 1, n):
            diff = arr[j] - arr[i]
            if diff > max_diff:
                max_diff = diff
    return max_diff

def max_difference_nlogn(arr):
    """
    O(n log n) solution using sorting.

    Sorts the array in ascending order.  The maximum difference will be the difference between the last element 
    and the first element. However, this doesn't guarantee j > i.  We need to find the *first* occurrence of the 
    minimum element and the *last* occurrence of the maximum element to ensure j > i.

    Args:
        arr: The input array of integers.

    Returns:
        The maximum difference arr[j] - arr[i] such that j > i, or 0 if no positive difference exists.
    """
    n = len(arr)
    if n < 2:
        return 0

    sorted_arr = sorted(arr)
    min_val = sorted_arr[0]
    max_val = sorted_arr[-1]

    # Find the first index of the minimum value
    first_min_index = -1
    for i in range(n):
        if arr[i] == min_val:
            first_min_index = i
            break

    # Find the last index of the maximum value
    last_max_index = -1
    for i in range(n - 1, -1, -1):
        if arr[i] == max_val:
            last_max_index = i
            break
    
    if first_min_index == -1 or last_max_index == -1:
        return 0

    if last_max_index > first_min_index:
        return max_val - min_val
    else:
        return 0

def max_difference_optimal(arr):
    """
    Optimal O(n) solution.

    Iterates through the array once, keeping track of the minimum element seen so far.
    For each element, calculates the difference between the current element and the minimum element.
    Updates the maximum difference if a larger difference is found.

    Args:
        arr: The input array of integers.

    Returns:
        The maximum difference arr[j] - arr[i] such that j > i, or 0 if no positive difference exists.
    """
    n = len(arr)
    if n < 2:
        return 0

    min_so_far = arr[0]
    max_diff = 0
    for i in range(1, n):
        diff = arr[i] - min_so_far
        if diff > max_diff:
            max_diff = diff
        if arr[i] < min_so_far:
            min_so_far = arr[i]

    return max_diff
