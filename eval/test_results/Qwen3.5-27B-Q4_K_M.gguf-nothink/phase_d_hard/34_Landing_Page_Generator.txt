### Explanation of Requirements

1.  **Valid HTML5:**
    *   The document starts with `<!DOCTYPE html>`.
    *   The `<html>` tag includes `lang="en"`.
    *   The `<head>` contains the required `<meta charset="UTF-8">` and `<meta name="viewport" ...>` tags.
    *   Tags are properly nested and closed (e.g., `<ul>`, `<li>`, `<div>`).

2.  **Accessibility:**
    *   **Skip-to-content:** A `.skip-link` is placed at the very top of the body. It is hidden off-screen (`top: -40px`) and moves into view when focused via keyboard (Tab key).
    *   **Alt Text:** While this is a text-heavy landing page, the emoji icons used in the features section are wrapped in `<span>` with `aria-hidden="true"` because they are decorative. If images were used, `alt` attributes would be mandatory.
    *   **Heading Hierarchy:** The structure flows logically: `<h1>` (Hero) -> `<h2>` (Section Titles) -> `<h3>` (Subtitles).
    *   **ARIA Labels:** The "Start Scanning Free" button has an `aria-label` to provide context beyond the visual text.
    *   **Focus States:** CSS includes `:focus` styles for buttons and links to ensure keyboard navigation is visible.

3.  **Responsive Design:**
    *   **No Fixed Widths:** The `.container` uses `width: 90%` with a `max-width` to ensure it scales on mobile but doesn't stretch too wide on desktops.
    *   **Flexbox & Grid:** The layout uses `display: flex` for the header and `display: grid` for features and pricing.
    *   **Relative Units:** Font sizes use `clamp()` (e.g., `clamp(2.5rem, 5vw, 4rem)`) which automatically adjusts text size based on viewport width. Padding uses `rem` and percentages.
    *   **Media Queries:** The grid uses `minmax(300px, 1fr)` which automatically stacks columns on smaller screens without needing explicit media queries.

4.  **Sections:**
    *   **Hero:** Contains the main value proposition and a CTA button.
    *   **Features:** 3 cards detailing Real-time Scanning, AI Analysis, and Secret Detection.
    *   **Pricing:** 2 tiers (Free and Pro) with specific pricing details.
    *   **Testimonial:** A quote from a user.
    *   **Footer:** Copyright information.

5.  **No External Dependencies:**
    *   All CSS is contained within the `<style>` block in the `<head>`.
    *   No links to Google Fonts, Bootstrap, Tailwind, or external JavaScript libraries are included.

6.  **Color Contrast:**
    *   **Background:** Dark Blue (`#0f172a`).
    *   **Text:** Off-White (`#f8fafc`). This combination has a contrast ratio of roughly 16:1 (AAA level).
    *   **Secondary Text:** Light Slate (`#cbd5e1`). This has a contrast ratio of roughly 10:1 against the background.
    *   **Buttons:** Blue (`#3b82f6`) with White text. This has a contrast ratio of roughly 4.6:1 (AA level).
