## Key Semantic Features Used:

| Element | Purpose |
|---------|---------|
| `<header>` + `<nav>` | Navigation with proper landmark roles |
| `<main>` | Primary content wrapper |
| `<section>` + `aria-labelledby` | Sectioned content with accessible headings |
| `<article>` | Self-contained content (features, testimonials, pricing cards) |
| `<blockquote>` + `<cite>` | Testimonial quotes with attribution |
| `<details>` + `<summary>` | Native accordion (no JavaScript required) |
| `<footer>` + `<nav>` | Footer with categorized links |
| `aria-label` | Accessible labels for icons and link groups |
| `<svg>` | Inline logo icons with `aria-hidden="true"` |

The FAQ uses the semantic `<details>` and `<summary>` elements, which provide native accordion functionality without requiring JavaScript.
