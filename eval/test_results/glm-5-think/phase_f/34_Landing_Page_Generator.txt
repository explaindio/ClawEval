## Key Semantic Choices Explained

| Element | Purpose |
|---------|---------|
| `<header>` | Contains the site navigation |
| `<nav>` | Identifies navigation regions (with `aria-label` for distinction) |
| `<main>` | Contains the primary page content |
| `<section>` | Groups thematically related content with `aria-labelledby` |
| `<article>` | Self-contained content (features, pricing tiers, testimonials) |
| `<aside>` | Supplementary content (trust badges) |
| `<blockquote>` | Semantic wrapper for testimonials |
| `<figure>` + `<figcaption>` | Groups testimonial images with attribution |
| `<details>` + `<summary>` | Native HTML accordion for FAQ |
| `<cite>` | Credits the source of a quote |
| `<address>` | Contact information in footer |
| `<small>` | Fine print / legal text |

The heading hierarchy follows a logical order: `h1` (page title) → `h2` (section titles) → `h3` (sub-sections/items).
